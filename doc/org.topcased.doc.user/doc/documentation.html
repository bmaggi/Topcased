Documentation

A set of 'Getting started' and 'tutorials' are available on this page :

  • Getting Started : Generate a graphical Editor : this getting started explains the generation process : how from a starting metamodel, we can generate a graphical editor with Topcased.
  • Getting Started : Generate a Tabbed Properties View : this tutorial helps you to create custom properties view featuring tabs and better graphic interfaces
  • Getting Started : Generate an Outline Menu : this tutorial helps you to configure and generate custom menus for your outline view

  • Tutorial : Network Editor : this tutorial helps you to create a simple metamodel with EMF and generate the associated graphical editor.
    • Editor Configuration : changes from 0.8.0 to 0.10.0 : this tutorial explains how to convert an editor configuration from TOPCASED v0.8.0 to v0.10.0.
    • Editor Configuration : changes from 0.9.0 to 0.10.0 : this tutorial explains how to convert an editor configuration from TOPCASED v0.9.0 to v0.10.0.

  • Documentation for the scripting support: how to run Topcased scripts and write your own.
  • SimplePDL : un langage de description de procédé.This tutorial propose a complete approach of the way to generate a graphical editor in an incremental way (the documentation is in French).
